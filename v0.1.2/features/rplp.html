compute plps.

Parameters

  • sig (array) ‚Äì a mono audio signal (Nx1) from which to compute features.

  • fs (int) ‚Äì the sampling frequency of the signal we are working with. Default is 16000.

  • num_ceps (float) ‚Äì number of cepstra to return. Default is 13.

  • pre_emph (int) ‚Äì apply pre-emphasis if 1. Default is 1.

  • pre_emph_coeff (float) ‚Äì apply pre-emphasis filter [1 -pre_emph] (0 = none). Default is 0.97.

  • win_len (float) ‚Äì window length in sec. Default is 0.025.

  • win_hop (float) ‚Äì step between successive windows in sec. Default is 0.01.

  • win_type (float) ‚Äì window type to apply for the windowing. Default is ‚Äúhamming‚Äù.

  • modelorder (int) ‚Äì order of the model / number of cepstra. 0 -> no PLP. Default is 13.

  • normalize (int) ‚Äì if True apply normalization. Default is 0.

compute rasta Perceptual Linear Prediction coefficients (rasta plp). [cepstra, spectra, lpcas] = rastaplp(samples, sr, do_rasta, modelorder)

Parameters

  • x (array) ‚Äì signal array.

  • fs (int) ‚Äì sampling rate. Default is 1600.

  • win_len (float) ‚Äì length of window. Default is 0.025,

  • win_hop (float) ‚Äì window hop. Default is 0.010,

  • do_rasta (bool) ‚Äì if True apply rasta filtering. If False PLP is calculated. Default is True,

  • modelorder (int) ‚Äì order of the model / number of cepstra. 0 -> no PLP. Default is 13,

  • normalize (int) ‚Äì if True apply normalization. Default is 0

Returns

PLP or RPLP coefficients. Matrix of features, row = feature, col = frame.

Example:

import scipy.io.wavfile
import spafe.utils.vis as vis
from spafe.features.mfcc import plp


#read wave file

# compute plps
