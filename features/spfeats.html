spafe.features.spfeats.spectral_flux(spectrum: numpy.ndarray, p: int = 2) → float[source]#

Compute the spectral flux, which measures how quickly the power spectrum of a signal is changing. This implementation computes the spectral flux using the L2-norm per default i.e. the square root of the sum of absolute differences squared [Scheirer].

Parameters

  • spectrum (numpy.ndarray) ‚Äì signal spectrum.

  • p (int) ‚Äì norm type. (Default is 2).

Returns

spectral flux.

Return type

(float)

Note

\[S_{flux} = (\sum_{k}(|a_{k}(t) - a_{k}(t-1))^2}{\sqrt{\sum_{k}a_{k}(t-1)|^p)^{\frac{1}{p}}\]

References

Scheirer(1,2)

: Scheirer, E., and M. Slaney. ‚ÄúConstruction and Evaluation of a Robust Multifeature Speech/Music Discriminator.‚Äù 1997 IEEE International Conference on Acoustics, Speech, and Signal Processing, 1997.

spafe.features.spfeats.extract_feats(sig: numpy.ndarray, fs: int, nfft: int = 512) → spafe.features.spfeats.SpectralFeats[source]#

Compute various spectral features [Peeters].

Parameters

  • sig (numpy.ndarray) ‚Äì input mono signal.

  • fs (int) ‚Äì input signal sampling rate.

  • nfft (int) ‚Äì number of FFT points. (Default is 512).

Returns

a dictionary including various frequency and spectral features (see Notes).

Return type

(dict)

Note

The resulting dictionary includes the following elements:

  • spectral features

    • spectral_centroid : spectral centroid.

    • spectral_skewness : frequencies skewness.

    • spectral_kurtosis : frequencies kurtosis.

    • spectral_entropy : spectral entropy.

    • spectral_spread : spectral spread.

    • spectral_flatness : spectral flatness.

    • spectral_rolloff : spectral rolloff.

    • spectral_flux : spectral flux.

    • spectral_mean : spectral mean.

    • spectral_rms : spectral root mean square.

    • spectral_std : spectral standard deviation.

    • spectral_variance : spectral variance.

Examples

from spafe.features.spfeats import extract_feats
from scipy.io.wavfile import read
from pprint import pprint

# read audio
fpath = "../../../tests/data/test.wav"
fs, sig = read(fpath)

# compute erb spectrogram
spectral_feats = extract_feats(sig, fs)
pprint(spectral_feats)
